# Implementation Plan: Kider Cleanup and Full Port of Neobrutalist Child-Friendly UI

This plan outlines the phase-by-phase migration of the **Genius Kids - Quiz Guidebook** student-facing frontend from the old Bootstrap-based "Kider" template to the new, gamified, child-friendly neobrutalist UI generated by Lovable (from [genius-kids.html](file:///c:/xampp/htdocs/Lovable/genius-kids-play-main/public/genius-kids.html)).

## User Review Required

> [!IMPORTANT]
> - **Removal of Bootstrap:** We will be replacing the Bootstrap stylesheet in the student layout with Tailwind CSS. Since Laravel Breeze auth screens and PWA styling already run on Tailwind, compiling the entire student UI with Tailwind v3 is cleaner and fits the neobrutalist style perfectly.
> - **Unused Routes Cleanup:** The Kider-specific pages (`about`, `classes`, `facility`, `team`, `testimonial`, `contact`) are being deleted as they are currently unrouted and represent dead code in the project.

## Decisions & Constraints

> [!NOTE]
> - **Sound Effects:** We will use the browser-synthesized Web Audio API sound effects (`tone()` logic) as implemented in the Lovable prototype.
